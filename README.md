# Guti Perú Studio — Fotografía & Audiovisual en Lima

Este proyecto es la **Landing Page oficial de Guti Perú Studio**, una plataforma de producción visual de alta gama diseñada por un fotógrafo profesional para materializar visiones artísticas.

---

## 🚀 Inicio Rápido
**Requisitos Previos:** Node.js 18+ y npm/yarn.

1.  **Instalar dependencias:** `npm install`
2.  **Modo desarrollo:** `npm run dev`
3.  **Construcción para producción:** `npm run build`
4.  **Iniciar servidor local:** `npm run start`

---

## 🏗️ Stack Tecnológico
*   **Framework:** [Next.js 15+](https://nextjs.org/) (App Router).
*   **Lenguaje:** [TypeScript](https://www.typescriptlang.org/) (Modo estricto).
*   **Estilos:** [Tailwind CSS](https://tailwindcss.com/) (Mobile-first).
*   **Animaciones:** [Framer Motion](https://www.framer.com/motion/).
*   **Iconos:** [Lucide React](https://lucide.dev/).

---

## 🧠 Arquitectura de la Información
El proyecto está diseñado para ser una **Landing Page de alto rendimiento** (Core Web Vitals optimizados):
*   `src/app`: Rutas del sistema, sitemaps y robots.
*   `src/components`: Componentes atómicos y secciones de UI.
*   `src/lib`: Constantes y utilidades centralizadas.
*   `public`: Activos estáticos (imágenes y logos).
