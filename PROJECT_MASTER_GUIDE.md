# Guti Perú Studio — Project Master Guide (Digital Identity & Brand Blueprint)

Esta es la guía maestra del proyecto **Guti Perú Studio**. Este documento está diseñado para servir como la **"Fuente Única de Verdad" (Single Source of Truth)** para desarrolladores, agencias de diseño y modelos generativos de IA encargados de crear presentaciones o material visual.

## 2. Guía Visual (Sistema de Diseño)
*   **Estética:** Minimalismo Moderno (estilo editorial de revista de lujo, Apple/Airbnb). Prioriza el "Negative Space" y la tipografía elegante.
*   **Paleta de Colores (Power Palette):**
    *   **Primario (Dorado/Acento):** `#b8965a` — Representa prestigio, luz y profesionalismo.
    *   **Fondo Principal (Oscuro):** `#131313` — Aporta profundidad y elegancia cinematográfica.
    *   **Fondo Secundario (Crema/Papel):** `#F5F0E8` — Usado en secciones de biografía para dar "aire" y calidez.
    *   **Texto Principal:** `#FFFFFF` (Blanco) y `#d1c5b5` (Crema suave).
*   **Tipografía (Google Fonts):**
    *   **Headlines (Títulos):** *Newsreader* (Serif). Estilo clásico, editorial, a menudo usado en cursiva para acentuar elegancia.
    *   **Body (Cuerpo):** *Inter* (Sans-serif). Limpia, moderna y de alta legibilidad.
    *   **Labels (Etiquetas/Botones):** *Montserrat*. Tracking amplio (espaciado entre letras) para un look de marca de lujo.

## 4. Stack y Excelencia Técnica
*   **Estructura de Software:** Next.js 15 (App Router), TypeScript (Modo estricto), Tailwind CSS.
*   **Despliegue:** 
    *   **Hosting:** Vercel (Optimizado para Web Vitals).
    *   **Infraestructura de Dominio:** Cloudflare (Registro y DNS de seguridad).
    *   **Dominio Oficial:** `https://gutiperu.com`
*   **Estrategia SEO:**
    *   Implementación de etiquetas canónicas y metadatos E-E-A-T (Experiencia, Especialización, Autoridad y Confianza).
    *   Arquitectura de Silos: Base sólida para futuras extensiones en subcarpetas (proyectos futuros: `/presets`, `/estudio`, `/talleres`).
